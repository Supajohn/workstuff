# h1 - Hello World

X) - Ensimmäisessä tekstissä kerrotaan SSH toiminta ja miten se ladataan sekä käyttö
- SSH avaimella poistetaan salasanan käyttö kun kirjaudutaan palvelimeen.
- Toinen artikkeli kertoo Ansiblesta.
- Se on työkalu jolla hallinnoidaan monta konetta samanaikaisesti. Eri tiedostot kertovat mitä siellä hallitaan.

A) Ensin SSH palvelimen asennusta komennolla - sudo apt install openssh-server
Tämän jälkeen voi katsoa Demonin tilan komennolla - sudo systemctl status ssh

B) Komento SSH-keygen ja sillä saa julkisen avaimen.
Tämä kopioidaan komennolla - ssh-copy-id localhost / suoraan orjakoneeseen ja pääsee ilman salasanaa SSH:lla.

C) Ensin ladataan Debianiin ansible eli komento - sudo apt-get install ansible
Tämän jälkeen oma hakemisto ansible - mkdir ansible
Seurataan Teron ohjeita ja luodaan ansibleen rakenne. 
- cd ansible
- nano hosts.ini  / tänne lisätään localhost tiedostoon
Seuraavaksi Ansible heruaa python ongelmaa joka ratkaistaa lisäämällä hosts.ini tiedostoon Teron antamaa koodia.

Eli - [all:vars]
    - ansible_python_interpreter=/usr/bin/python3

Tämän jälkeen ei tule python erroria.

Pitää alkaa luomaan eri rooleja.

Site.yml ohjaa mitä konfigurointeja ajetaan millekin koneelle. 
Tämän sisällöksi tulee 
- hosts: all
  roles:
    - hello_world

Seurataan Teron ohjeita vielä tarvittavien tiedostojen kanssa. Elikkäs seuraavat komennot
- mkdir -p roles/hello_world/tasks
nano roles/hello_world/tasks/main.yml
- copy:
    dest: /tmp/hello-world
    content: "Hello World!"
